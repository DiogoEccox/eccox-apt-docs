O serviço **Typing for BND File** permite a análise de arquivos `.bnd` utilizados por aplicativos em plataforma distribuída que executam comandos SQL estáticos no ambiente Mainframe via **DDF (Distributed Data Facility)**.

Essa funcionalidade é exclusiva da feature **ECCOX APT® EDGE APPLICATIONS FOR DDF**, e só estará disponível caso o usuário possua acesso a essa feature específica.

## Como utilizar

1. Clique no botão **TYPING FOR BND FILE** na interface do sistema.
2. Na pop-up exibida (Figura 34), clique no campo **SELECT FILE**.
3. Selecione o arquivo com extensão `.bnd` localizado em seu ambiente local.
4. Após a seleção, clique no botão **UPLOAD BND FILE** que será habilitado.

O serviço utilizará o arquivo para identificar o programa e os relacionamentos associados a ele.

> **Nota:** Esse recurso permite o uso de **bases DB2 clonadas e isoladas**, em substituição às bases originais.
