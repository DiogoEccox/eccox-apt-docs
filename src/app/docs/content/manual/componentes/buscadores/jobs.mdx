O serviço **JOBS** permite adicionar ao container os programas e bases de dados identificados a partir de um **JOB**.

A busca é feita com base no nome do JOB e na biblioteca de origem informada. A análise retorna os programas e bases envolvidos na execução do JOB.

### Campos do Formulário

- **Origin Type**: Origem do JOB. Atualmente, a opção disponível é `Library`.
- **JOB Name**: Nome do JOB.
- **DSNAME**: Nome da biblioteca de origem do JOB.

Para executar:

1. Preencha os dados do formulário.
2. Clique no botão **EXECUTE** para iniciar o processamento.
