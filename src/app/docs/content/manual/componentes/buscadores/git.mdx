O serviço **GIT** permite acessar um repositório de dados e obter os programas com base nas informações fornecidas pelo usuário.

### Campos do Formulário

- **Organization Name**: Selecione a organização cadastrada. Ao selecionar, o campo "Project Name" será carregado com os projetos da organização.
- **Project Name**: Projeto que será utilizado como origem. Ao selecionar, o campo "Repository Name" será carregado com os repositórios do projeto.
- **Repository Name**: Repositório que deseja acessar. A seleção carregará o campo "Branch Name" com os branchs disponíveis.
- **Branch Name**: Branch que será utilizado para obter os programas associados.

Após o preenchimento das informações, clique no botão **PROCESS** para executar o serviço.
