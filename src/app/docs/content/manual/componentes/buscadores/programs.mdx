O serviço **Programs** permite identificar bases **DB2** acessadas por um programa e adicioná-las ao container de teste.

A análise considera os acessos mapeados no DB2 para o programa informado.

### Campo do Formulário

- **Program**: Nome do programa a ser analisado

Para adicionar uma nova busca:

1. Preencha o campo com o nome do programa.
2. Clique em **NEW** para adicionar à lista de processamento.
3. Pressione **EXECUTE** para iniciar o serviço.
