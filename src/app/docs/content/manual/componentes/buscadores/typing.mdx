### Campos do Formulário

- **Program**: Nome do objeto (programa).
- **Type**: Tipo da biblioteca de origem, podendo ser:
  - `LOADLIB`
  - `DBRMLIB`
  - `PSBLIB`
- **DSNAME**: Nome da biblioteca de origem do objeto.

### Procedimento

1. Preencha os campos obrigatórios.
2. Clique no botão **NEW** para adicionar a requisição na lista de processamento.
3. Pressione **EXECUTE** para iniciar a busca.
