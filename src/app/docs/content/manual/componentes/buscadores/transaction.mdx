O serviço de **Transaction** busca, a partir de uma transação fornecida, o nome do programa relacionado para inclusão no container.

### Campos do Formulário

- **Type**: Tipo da transação informada. Pode ser **CICS** ou **IMS**
- **Code**: Código da transação
- **Discovery**: Retorna o programa correspondente à transação e toda a Cross-reference associada

### Como utilizar

1. Preencha os dados obrigatórios
2. Clique no botão **NEW** para adicionar a requisição à lista de processamento
3. Pressione **EXECUTE** para iniciar o serviço
