O serviço **DLI/DB** permite adicionar **DBDs (DataBase Description)** do **IMS** ao container e identificar os programas que acessam esse DBD. Caso identificados, os programas podem ser adicionados automaticamente junto ao DBD no container.

Esse serviço utiliza acessos montados no **catálogo do IMS** para identificar as dependências entre programas e bases de dados.

### Campos do Formulário

- **DBD Name**: Nome do DBD IMS a ser pesquisado.

### Como utilizar

1. Preencha o nome do DBD desejado.
2. Clique em **NEW** para incluir a requisição na lista de processamento.
3. Pressione **EXECUTE** para iniciar a execução do serviço.
