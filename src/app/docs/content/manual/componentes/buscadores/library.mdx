O serviço **Library** permite buscar programas a partir de um **DSNAME** de uma biblioteca de fontes e um **prefixo de pesquisa**.

A análise retorna todos os programas que correspondem ao prefixo informado. Por exemplo:

- Prefixo `SHP*` → busca por todos os programas que começam com `SHP`
- Prefixo `SHPPROGM` → busca exata por `SHPPROGM`

Opcionalmente, pode-se solicitar uma **XREF**, que permite incluir no container programas ou bases relacionadas aos encontrados inicialmente.

### Campos do Formulário

- **DSNAME**: Nome da biblioteca de fontes
- **Prefix**: Prefixo de pesquisa dos programas
- **DISCOVERY**: Solicita XREF para os programas encontrados

### Como Utilizar

1. Preencha os campos solicitados.
2. Clique em **NEW** para incluir a requisição na lista de processamento.
3. Pressione **EXECUTE** para iniciar a busca.
