A adição e uso de componentes em um container se divide em duas etapas principais:

- **Busca e seleção dos componentes**, realizada por meio dos **serviços buscadores**.
- **Processamento**, feito através dos **botões de ações**.

> Todo componente adicionado ou atualizado precisa passar por um processamento. A seleção feita pelos serviços buscadores representa uma definição lógica, enquanto o processamento consolida as ações parametrizadas e gera os clones dos componentes para testes.

Para adicionar componentes:

1. Acesse o menu **Test Container > Browse**.
2. Localize o container desejado.
3. Clique no botão <Pencil className="inline w-4 h-4 mx-1 text-muted-foreground" /> para editá-lo.
