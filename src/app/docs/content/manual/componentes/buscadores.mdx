> **Nota:** Os botões dos serviços buscadores podem variar de acordo com as features contratadas e os acessos de cada usuário.

Esses serviços facilitam a busca e seleção de componentes, permitindo montar o container com agilidade. Para utilizar:

1. Clique no botão referente ao serviço desejado.
2. Forneça os dados para entrada da pesquisa.
3. Visualize a lista de componentes retornados.

### Origem dos Componentes

Os componentes retornados podem ter origem:

- **REM (Remoto):** Biblioteca de outra LPAR.
- **LOC (Local):** Biblioteca da LPAR de origem.
- **PRD (Produção):** Bibliotecas configuradas como produção.

Cada origem é identificada conforme o comportamento do serviço buscador e a customização do ambiente.

### Substituição de Versão

Caso um componente já esteja presente no container e seja necessário atualizá-lo:

- Execute novamente o serviço buscador correspondente.
- Solicite a substituição do componente encontrado.
